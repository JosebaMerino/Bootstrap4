Button

Simplemente botones que tienen un estilo bastante diferente a los comunes

La clase btn se le puede aplicar a otros elementos que no sean button como por ejepmlo a e input

Link

Los botones tambien puede tener un estilo diferente llamado outline. Y viene uno para cada color.

Tambien se le puede cambiar el tamaño de los botones. Tienen dos tamaños a parte del original: btn-lg y btn-sm

Se puede hacer que los botones ocupen todo el ancho del padre con la clase btn-block

Los botones pueden tener dos estados: activos o inactivos. Por defecto se encuentran en estado activo pero si queremos cambiar el estado, bastaria con aplicar el atributo disabled para deshabilitarlo. Si de lo contrario se desea habilitar, se usará la clase active
